# Sedona Project

Sedona — это проект одностраничного сайта для изучения и закрепления навыков адаптивной вёрстки и работы с интерфейсами. Основная цель — создать современный интерфейс, соответствующий макетам и техническому заданию.

## Проект можно посмоттреть по ссылке:
[https://rnkuchin.github.io/edu-project-sedona/](https://rnkuchin.github.io/edu-project-sedona/)

## Описание проекта

Sedona включает в себя:
- Главную страницу с информацией о городе, привлекательными блоками и элементами интерфейса.
- Страницу каталога с фильтрацией, сортировкой и списком товаров.

Проект разработан с учётом базовых принципов веб-дизайна и веб-стандартов, таких как:
- **Чистый HTML и CSS**, без использования сторонних фреймворков.
- **Прогрессивное улучшение**, что делает сайт доступным для современных браузеров.
- Кроссбраузерная поддержка (Chrome, Firefox, Safari).

## Основные функции

### Главная страница
- Шапка сайта с элементами навигации и интерактивными кнопками.
- Крупный баннер с контентным изображением и наложением.
- Интерактивные блоки (жильё, еда, сувениры) с декоративными иконками.
- Формы подписки с проверкой заполнения полей перед отправкой.
- Модальные окна (в демо реализованы как заглушки).

### Страница каталога
- Фильтр товаров с рендж-слайдером и кнопками для применения и сброса настроек.
- Сортировка и смена видов отображения каталога с использованием GET-параметров.
- Карточки товаров с элементами управления (в избранное, подробнее).

## Особенности

- **Шрифты**: Используется шрифт PT Sans из Google Fonts.
- **Дизайн**: Соответствует макету и стайлгайду.
- **Максимальная консистентность**: Все расхождения между макетом и стайлгайдом трактуются в пользу стайлгайда.
- **Элементы взаимодействия**: Логотипы и кнопки корректно ведут на целевые страницы или открывают модальные окна.

## Как запустить проект

1. Склонируйте репозиторий:
   ```git clone https://github.com/your-username/sedona.git```
   
2. Откройте файл index.html в любом современном браузере.
